🚀 Project Overview

This project, "Initialize a Git Repository", automates the process of creating a new Git project folder using a simple Bash script.

👍 For detailed instructions on running and testing the project, see USAGE.md →

The main script (create_repo.sh) performs the following tasks automatically:

  • 🗇️ Creates a new folder named new_project on the user’s Desktop.
  • ⚙️ Enters the folder and initializes a Git repository using git init.
  • 📝 Creates a basic README.md file with project information.
  • 🗝️ Adds all files and commits them with the message “Initial commit.”

This saves time for users who frequently set up new projects, ensuring a consistent and clean starting point for version control.
