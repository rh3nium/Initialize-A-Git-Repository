# Unit 3 - Bash Git Repo Setup/Initialization Script

This simple Bash script creates a folder named `new_project` on the Desktop and initializes a Git repository inside it.

## Usage

```bash
bash create_repo.sh
```

This script can be useful in the following situations:

- Class Assignments: Quickly set up Git repositories for coursework or lab exercises.

- Learning Git: Ideal for beginners who want to understand how repositories are initialized.

- Automation: Saves time by automating repetitive folder and Git setup commands.

- Project Organization: Ensures consistent folder structures for multiple small projects.

- Testing Git Commands: Developers can use it to experiment safely without affecting important repos.
